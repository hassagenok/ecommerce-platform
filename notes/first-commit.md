# Первый локальный commit

Я сохраняю в истории не всю папку автоматически, а выбранное состояние файлов.
Сначала Git показывает изменения через status, потом я добавляю нужные файлы в staging area, затем создаю commit.
Commit пока остается только на моей машине и не отправляется в origin. Working tree is the place where I edit files before staging. Staging area can hold an older prepared version than my working tree.
