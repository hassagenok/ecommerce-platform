# PhpStorm и CLI проверяют один Git repository

PhpStorm показывает changed files, но я проверяю их через make status.
PhpStorm показывает diff, но я умею открыть тот же смысл через git diff.
Окно commit помогает выбрать файлы, но commit остается обычным Git commit.
После commit я проверяю make status, make log и make check.
